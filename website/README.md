# אתר תצוגה — שמחות פלוס

אתר ניווט בעברית (RTL) שמציג את כל נתוני התוכנית העסקית והמצגת של שמחות פלוס בתוך ממשק אחד — במקום קובץ אקסל בלבד.

## מה כולל האתר

- **תפריט צד עם 15 גיליונות**: מבוא, סקר מתחרים, מיפוי אולמות, נתוני שוק, מגזרים וסכומי מתנות, סקר לקוחות, סקר אולמות B2B, תוכנית עסקית, רווח והפסד, ניתוח עמדה, תחזית 7 שנים, רעיונות וחדשנות, ממשק והרחבות, תוכנית עבודה ומקורות.
- **מיפוי 342 אולמות**: כרטיסי מדדים, סינון לפי קטגוריה וחיפוש חי לפי שם, עיר או אזור.
- **טבלאות פיננסיות**: מטריצות רווח והפסד ותחזיות עם מספרים מיושרים.
- **מציג מצגת מובנה**: כל 25 השקפים כתמונות, עם לחיצה להגדלה וגלישה בין השקפים.
- **מצב כהה/בהיר**, עיצוב רספונסיבי למובייל, באנרים ואנימציות עדינות.

## הפעלה מקומית

אתר סטטי בלבד (HTML/CSS/JS, ללא תלויות build). להרצה מקומית:

```bash
cd website
python3 -m http.server 8000
# ואז לפתוח http://localhost:8000
```

## מבנה הקבצים

```
website/
├── index.html          # שלד האתר
├── base.css            # reset בסיסי
├── style.css           # מערכת עיצוב (פלטת מותג: טורקיז כהה + זהב)
├── app.js              # לוגיקת ניווט, טבלאות, סינון אולמות ומציג שקפים
└── assets/
    ├── data.json       # כל נתוני 15 הגיליונות
    └── slides/         # 25 שקפי המצגת כתמונות
```

הנתונים נטענים מתוך `assets/data.json` שהופק מקובץ האקסל `business-plan/SimchotPlus_business_plan_expanded.xlsx`.
